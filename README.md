# Tomer Atia - GitHub Pages Portfolio

This repository hosts my personal portfolio site (GitHub Pages).

## Live Site
https://tomeratia.github.io

## Structure
- `index.html` - main page
- `style.css` - styles
- `script.js` - links + contact form mailto logic
- `assets/` - images (profile + project previews)

## Update Images
Replace these files (keep the same filenames):
- `assets/profile.jpg`
- `assets/stormvision.jpg`
- `assets/seaalert.jpg`
- `assets/cornealseg.jpg`

## Deploy
GitHub Settings -> Pages:
- Source: Deploy from a branch
- Branch: `main`
- Folder: `/(root)`
